# voice_chatbot_practice

# 🚞 국내 관광지 추천기 🏄

음성이나 텍스트로 여행 조건을 입력하면,  
LLM이 조건에 맞는 국내 관광지 3곳을 추천해주고  
추천 결과를 음성으로 다시 들려주는 Streamlit 프로젝트입니다.

---

## 1. 프로젝트 소개

이 프로젝트는 사용자가 원하는 여행 조건을 음성 또는 텍스트로 입력하면,  
해당 조건에 맞는 국내 관광지를 추천해주는 간단한 웹 애플리케이션입니다.

예를 들어,

- 바다가 보이는 곳
- 사람이 적은 곳
- 수도권에서 가까운 곳

같은 조건을 입력하면, OpenAI 모델이 이를 바탕으로 국내 관광지 3곳을 추천합니다.  
또한 추천 결과를 TTS로 변환하여 음성으로도 들을 수 있습니다.

---

## 2. 주요 기능

### 1) 음성 입력
- 사용자가 마이크를 통해 여행 조건을 말하면 텍스트로 변환합니다.
- `speech_recognition` 라이브러리를 사용합니다.

### 2) 텍스트 입력
- 사용자가 원하는 조건을 직접 입력할 수 있습니다.
- 음성 입력과 텍스트 입력을 함께 사용할 수 있습니다.

### 3) 관광지 추천
- 입력된 조건을 바탕으로 OpenAI API를 활용해 국내 관광지 3곳을 추천합니다.
- 각 관광지에 대해 추천 이유 2가지와 평점을 제공합니다.

### 4) 음성 출력
- 추천 결과를 `gTTS`를 이용해 mp3 음성으로 변환합니다.
- Streamlit 화면에서 바로 재생할 수 있습니다.
